# FSD50K Audio Robustness Project

## Team
- Jaime Cano Morano
- Ilias Mahfoud Benhaddou

## Project Goal
Build a sound recognition system using the FSD50K dataset and study how robust it is under noisy and imperfect audio conditions.

## Main Task
Multi-label audio classification: each clip may contain more than one sound category.

## Project Structure
- `data/` → dataset and metadata
- `notebooks/` → exploration and experiments
- `src/` → reusable code
- `models/` → saved models
- `results/` → figures, tables, outputs

## First Milestones
1. Set up the shared GitHub repo
2. Inspect the dataset and labels
3. Build a baseline model
4. Build a CNN model
5. Evaluate robustness under noise perturbations

## Notes
Do not upload the full dataset or heavy checkpoints to GitHub.
